# StudyMate Pro

StudyMate Pro is an Android-based academic assistance application developed using **Java and Firebase**.  
The app is designed to help students efficiently manage their academic activities by providing tools such as notes management, GPA calculation, CGPA calculation, and marks evaluation in a single platform.

## üèó System Architecture
- **Frontend:** Android UI designed using XML
- **Backend:** Firebase Authentication and Firestore Database

The application follows a modular architecture where each feature is implemented as a separate module, ensuring better maintainability and scalability.

## üì¶ Modules
